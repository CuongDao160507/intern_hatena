# Bài tập Tuần 2 - Contact Form Fullstack (MERN Stack)

**Họ và tên:** Đào Việt Cường
**Nhánh (Branch):** `cuongDao`
**Thư mục dự án:** `Project_Tuan02`

## 1. Giới thiệu

Đây là bài tập thực hành xây dựng ứng dụng web Fullstack cơ bản.

- **Frontend:** ReactJS (Vite) - Xây dựng giao diện Form liên hệ.
- **Backend:** NodeJS & Express - Tạo API nhận dữ liệu từ Client.
- **Database:** MongoDB Atlas - Lưu trữ thông tin người dùng gửi lên.

## 2. Công nghệ sử dụng

- **Frontend:** ReactJS, Axios, CSS.
- **Backend:** NodeJS, Express, Mongoose, Cors.
- **Database:** MongoDB (Cloud Atlas).
